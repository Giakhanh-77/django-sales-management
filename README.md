# Django Sales Management System

## Giới thiệu
Dự án web quản lý bán hàng đa nền tảng được xây dựng bằng Django.

Hệ thống hỗ trợ:
- Quản lý sản phẩm
- Quản lý đơn hàng
- Quản lý khách hàng
- Quản lý kho hàng
- Báo cáo doanh thu

## Công nghệ sử dụng
- Python
- Django
- HTML / CSS / Bootstrap
- MySQL

## Cấu trúc hệ thống
- `products`: quản lý sản phẩm
- `orders`: quản lý đơn hàng
- `customers`: quản lý khách hàng
- `inventory`: quản lý kho
- `reports`: báo cáo thống kê
- `reviews`: đánh giá sản phẩm

## Cách chạy
```bash
python manage.py runserver
```
